Custom formulas

Add a metric using Excel-style functions. Use fields like {Crop Acres} or normalized fields like {crop_acres}. Formulas aggregate across all matching rows unless you specify a dataset filter.

Supported locally: SUM, AVG/AVERAGE, MIN, MAX, COUNT, COUNTIF, ROUND, ABS, IF, SAFE_DIVIDE, arithmetic operators, and field references in braces. When a browser can load an external formula engine later, this structure can be extended to full HyperFormula/Excel coverage.
